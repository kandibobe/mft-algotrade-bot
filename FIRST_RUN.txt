Открой PowerShell и выполни:

ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:

ШАГ 2: API КЛЮЧИ BINANCE (10 минут)
═══════════════════════════════════════════════════════════════════

1. Зайди на https://www.binance.com/

2. Нажми на иконку профиля → API Management

3. Создай новый API ключ:
   - Нажми "Create API"
   - Выбери "System Generated"
   - Введи метку (например: "Stoic Citadel Dry Run")

4. Настрой разрешения:
   
   ДЛЯ DRY-RUN ТЕСТИРОВАНИЯ:
   ✅ Enable Reading                    ← ВКЛЮЧИ
   ❌ Enable Spot & Margin Trading      ← НЕ ВКЛЮЧАЙ
   ❌ Enable Withdrawals                ← НИКОГДА!
   
   ДЛЯ LIVE TRADING (ПОЗЖЕ, ПОСЛЕ 2 НЕДЕЛЬ ТЕСТОВ):
   ✅ Enable Reading                    ← ВКЛЮЧИ
   ✅ Enable Spot & Margin Trading      ← ВКЛЮЧИ
   ❌ Enable Withdrawals                ← НИКОГДА!

5. Скопируй и сохрани:
   - API Key (длинная строка букв и цифр)
   - Secret Key (показывается ОДИН РАЗ!)
   
   ⚠️  ВАЖНО: Secret Key показывается только один раз!
       Скопируй его сразу и сохрани в безопасном месте!

6. Настрой IP Whitelist (опционально но рекомендуется):
   - Restrict access to trusted IPs only
   - Добавь свой IP адрес


ШАГ 3: НАСТРОЙКА .ENV (2 минуты)
═══════════════════════════════════════════════════════════════════

1. Открой .env файл:

   notepad .env

2. Заполни три строки:

   BINANCE_API_KEY=твой_api_key_сюда
   BINANCE_API_SECRET=твой_secret_key_сюда
   FREQTRADE_API_PASSWORD=придумай_свой_пароль_сюда

   Пример:
   BINANCE_API_KEY=Abc123xyz789DefGhi456Jkl
   BINANCE_API_SECRET=XyZ789AbC123MnO456PqR789StU
   FREQTRADE_API_PASSWORD=MySecurePassword2024!

3. Проверь что DRY_RUN=true:

   DRY_RUN=true    ← Должно быть true!

4. СОХРАНИ файл (Ctrl+S) и закрой


ШАГ 4: ПЕРВИЧНАЯ НАСТРОЙКА (5 минут)
═══════════════════════════════════════════════════════════════════

Выполни команды по порядку:

1. Setup (создаст нужные директории):

   .\stoic.ps1 setup

   Что происходит:
   - Проверяется Docker
   - Проверяется .env
   - Создаются директории
   
   ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
   ✅ Setup completed!

2. Скачай исторические данные:

   .\stoic.ps1 download-data

   Что происходит:
   - Скачиваются данные за 90 дней
   - Для пар: BTC/USDT, ETH/USDT, BNB/USDT и др.
   - Таймфреймы: 5m, 15m, 1h
   
   ⏱️  Это займет 3-5 минут
   
   ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
   ✅ Data download completed!


ШАГ 5: ЗАПУСК! (2 минуты)
═══════════════════════════════════════════════════════════════════

1. Запусти paper trading:

   .\stoic.ps1 trade-dry

   Что происходит:
   - Запускаются Docker контейнеры
   - Freqtrade начинает торговать (виртуально!)
   - FreqUI dashboard становится доступен
   
   ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
   ✅ Trading bot started (dry-run mode)!

2. Открой dashboard:

   .\stoic.ps1 dashboard

   Откроется браузер на http://localhost:3000
   
   Логин:  stoic_admin
   Пароль: твой_пароль_из_env

3. ГОТОВО! БОТ РАБОТАЕТ! 🎉

ЧАСТЫЕ ВОПРОСЫ
═══════════════════════════════════════════════════════════════════

Q: Сколько ждать первых сделок?
A: В dry-run режиме бот ждет подходящие условия для входа.
   Это может занять от нескольких минут до нескольких часов.
   Не паникуй - это нормально!

Q: Могу ли я потерять деньги?
A: В DRY_RUN режиме - НЕТ! Это виртуальная торговля.
   Все сделки "понарошку", никаких реальных денег.

Q: Когда переходить на live trading?
A: МИНИМУМ через 2 недели успешного paper trading.
   Следуй плану в DEVELOPMENT_PLAN.md

Q: Что делать если ошибка?
A: 1. Проверь .\health.ps1
   2. Посмотри логи: .\stoic.ps1 logs
   3. Прочитай QUICKSTART_WINDOWS.md
   4. Спроси в Issues на GitHub

Q: Нужен ли интернет?
A: Да, бот постоянно получает данные с биржи.
